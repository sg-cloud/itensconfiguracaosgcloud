Melhoria na carga de produtos para o PDV Delphi

Hoje utilizamos o agendador para enviar produtos para o PDV em determinados periodos ou a cada x tempos

Vamos utilizar o agendador, porém agora de forma manual. Vamos utilizar ele para marcar os produtos que tiveram alterações, porém ao invés de gerar a carga automatica, será criado um botão na tela de gerar carga "Geranciamento de produto" para gerar a carga dos produtos alterado.
